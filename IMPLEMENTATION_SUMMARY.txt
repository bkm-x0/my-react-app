🎯 ملخص المشروع:
════════════════════════════════════════════════════════════════════════════

تم إضافة نظام شامل ومتكامل لحساب الضرائب والفائدة بناءً على فرق السعر بين:
✓ سعر الشراء من المورد (التكلفة)
✓ سعر البيع للعميل (الإيراد)

🔐 مستويات الحماية:
════════════════════════════════════════════════════════════════════════════

1️⃣  المصادقة (Authentication)
    ✓ يتطلب Bearer Token صحيح
    ✓ التحقق من صحة JWT
    ✓ رفض أي طلب بدون token

2️⃣  التفويض (Authorization)
    ✓ يتطلب دور "admin" فقط
    ✓ رفض أي دور آخر
    ✓ فحص الدور على كل طلب

3️⃣  التدقيق (Auditing)
    ✓ تسجيل كل عملية إدمن
    ✓ تسجيل بيانات المسؤول
    ✓ تسجيل التاريخ والوقت والـ IP

🚀 البدء السريع:
════════════════════════════════════════════════════════════════════════════

1️⃣  تشغيل الهجرة (إنشاء الجداول):
    node backend/migrate-tax-system.js

2️⃣  اختبار الحسابات:
    node backend/tests/taxCalculator.test.js

3️⃣  اختبار الحماية:
    node backend/tests/taxAccessControl.test.js

4️⃣  تسجيل الدخول كمسؤول:
    curl -X POST http://localhost:5000/api/auth/login \
      -H "Content-Type: application/json" \
      -d '{"email": "admin@example.com", "password": "password"}'

5️⃣  استخدام النظام مع Token:
    curl -X POST http://localhost:5000/api/tax/calculate \
      -H "Authorization: Bearer {TOKEN}" \
      -H "Content-Type: application/json" \
      -d '{
        "costPrice": 100,
        "sellingPrice": 150,
        "quantity": 5,
        "taxRate": 0.15,
        "interestRate": 0.05
      }'

📊 رسائل الخطأ:
════════════════════════════════════════════════════════════════════════════

401 Unauthorized:
  "Not authorized, please login first"
  → المستخدم لم يقدم token

401 Unauthorized:
  "Not authorized, token expired or invalid. Please login again."
  → Token غير صحيح أو منتهي الصلاحية

403 Forbidden:
  "Not authorized as admin"
  → المستخدم ليس مسؤول
